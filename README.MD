## рҹҢҹ Overview

This project demonstrates a complete DevOps workflow for a full-stack e-commerce inventory app. It covers every stage of the software delivery lifecycle вҖ” from local development to containerization, automated testing, code quality analysis, infrastructure provisioning, and container orchestration.

## вңЁ Features

- вң… Full CRUD for product inventory (Create, Read, Update, Delete)
- вң… RESTful API with FastAPI and auto-generated Swagger docs
- вң… React TypeScript frontend with product cards
- вң… Dockerized backend and frontend
- вң… Automated Jenkins CI/CD pipeline
- вң… SonarQube code quality analysis
- вң… Terraform scripts for AWS EC2 provisioning
- вң… Ansible playbooks for server configuration
- вң… Kubernetes manifests for container orchestration

## рҹҡҖ Getting Started

### Prerequisites
- Git
- Docker Desktop
- Node.js v22+
- Python 3.13+

### Run Locally

**1. Clone the repo:**
```bash
git clone https://github.com/krishagr1451/Devops-project.git
cd Devops-project
```

**2. Start with Docker Compose:**
```bash
docker-compose up --build
```

**3. Access the app:**
- Frontend вҶ’ http://localhost:3000
- Backend API вҶ’ http://localhost:8000/docs

### Run Without Docker

**Backend:**
```bash
cd backend
python -m venv venv
venv\Scripts\activate        # Windows
pip install -r requirements.txt
uvicorn app.main:app --reload
```

**Frontend:**
```bash
cd frontend
npm install --legacy-peer-deps
npm start
```

## вҡҷпёҸ CI/CD Pipeline

The Jenkins pipeline runs automatically on every push to `main`:

```
Checkout вҶ’ SonarQube Analysis вҶ’ Docker Build вҶ’ Deploy
```

| Stage | Description |
|---|---|
| **Checkout** | Pulls latest code from GitHub |
| **SonarQube Analysis** | Scans code for bugs and vulnerabilities |
| **Build Docker Images** | Builds backend and frontend images |
| **Deploy** | Spins up containers via Docker Compose |

### Setup Jenkins
```bash
docker run -d --name jenkins -p 8080:8080 \
  -v jenkins_home:/var/jenkins_home \
  -v /var/run/docker.sock:/var/run/docker.sock \
  jenkins-docker
```

## рҹҢҚ Infrastructure (Terraform)

Terraform provisions the following AWS resources:
- **VPC** with public subnet
- **Internet Gateway** and Route Table
- **Security Group** (ports 22, 80, 3000, 8000)
- **EC2 Instance** (Ubuntu 22.04, t2.micro)

```bash
cd terraform
terraform init
terraform plan
terraform apply
```

```bash
kubectl apply -f k8s/backend-deployment.yml
kubectl apply -f k8s/frontend-deployment.yml
```
